## 3. 팩토리얼
*************************************************************************
### 설명

자연수 n이 입력되면 n!를 구하는 프로그램을 작성하시오. 
예를 들어 5! = 5*4*3*2*1 = 120 입니다. 

-------------------------------------------------------------------------
### 입력
첫 번째 줄에 자연수 n(1<=n<=100)이 주어집니다.

-------------------------------------------------------------------------
### 출력
첫 번째 줄에 n팩토리얼 값을 출력합니다. 

| 예시 입력 | 결과  |
|-------|-----|
| 5     | 120 |
-------------------------------------------------------------------------
