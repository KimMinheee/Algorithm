## 1. 재귀 함수(스택프레임)
*************************************************************************
### 설명

자연수 n이 입력되면 재귀함수를 이용하여 1부터 n까지를 출력하는 프로그램을 작성하시오. 

-------------------------------------------------------------------------
### 입력

첫 번째 줄은 정수 n(3<= n <= 10)이 입력된다. 

-------------------------------------------------------------------------
### 출력
첫째 줄에 출력한다. 

| 예시 입력 | 결과    |
|-------|-------|
| 3     | 1 2 3 |
-------------------------------------------------------------------------
